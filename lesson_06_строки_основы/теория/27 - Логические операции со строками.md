# Билет 27: Логические операции со строками, chr() и ord()

## Истинностность строки

Пустая строка — falsy, непустая — truthy (билет 12):

```python
bool("")        # False
bool("a")       # True
bool(" ")       # True — пробел это символ, строка непуста!

if name:        # питонично: "не пустая ли строка"
    ...
```

## Сравнение строк

Строки сравниваются **лексикографически** — посимвольно по **кодам Unicode**:

```python
print("apple" < "banana")   # True  ('a' < 'b')
print("apple" < "apply")    # True  (сравнение до 'e' < 'y')
print("Zoo" < "apple")      # True! 'Z'(90) < 'a'(97) — заглавные раньше
print("abc" == "abc")       # True
```

 (!) Все заглавные латинские буквы «меньше» строчных (их коды меньше). Для регистронезависимого сравнения приводят к одному регистру:

```python
print("Apple".lower() < "banana".lower())   # True
```

## Функции chr() и ord()

Мост между символом и его кодом Unicode:

- `ord(c)` — код символа (int);
- `chr(n)` — символ по коду.

```python
print(ord("A"))     # 65
print(ord("a"))     # 97
print(ord("я"))     # 1103
print(chr(65))      # 'A'
print(chr(97))      # 'a'
print(chr(0x1F600)) # '😀' — эмодзи по коду
```

Они **обратны** друг другу:

```python
print(chr(ord("Z")))   # 'Z'
print(ord(chr(1000)))  # 1000
```

## Применение chr/ord

Сдвиг символов (например, шифр Цезаря):

```python
def shift(ch, n):
    return chr(ord(ch) + n)

print(shift("a", 1))    # 'b'
print(shift("z", -25))  # 'a'

# Алфавит
alphabet = [chr(c) for c in range(ord("a"), ord("z") + 1)]
# ['a', 'b', ..., 'z']
```

(!) `ord` принимает **ровно один** символ:

```python
ord("ab")    # TypeError: ord() expected a character, but string of length 2
```

## Итоги

- `""` falsy, непустая строка truthy (даже `" "`);
- строки сравниваются **лексикографически по кодам Unicode**; заглавные < строчных;
- регистронезависимо — через `.lower()`/`.upper()`;
- `ord(c)→код`, `chr(n)→символ`, взаимно обратны;
- `ord` — только для одного символа.

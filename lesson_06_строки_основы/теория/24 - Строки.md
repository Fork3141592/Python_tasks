# Билет 24: Строки (str)

## Что такое строка

`str` — **неизменяемая** последовательность Unicode-символов.

```python
s = "Привет"
s[0] = "п"     # TypeError: 'str' object does not support item assignment
```

«Изменение» строки = создание новой:

```python
s = "abc"
s = s.upper()      # новый объект "ABC"
```

## Строковые литералы

```python
'одинарные'
"двойные"          # эквивалентны; выбор — ради кавычек внутри
"It's ok"          # двойные, чтобы внутри был апостроф
'Он сказал "да"'
```

Тройные кавычки — строка на несколько строк (сохраняет переводы строк):

```python
text = """Первая строка

Третья"""

# Часто используется как docstring
def f():

```

## str vs repr

Два строковых представления объекта:

| | `str(x)` | `repr(x)` |
|---|---|---|
| Для кого | для **человека** (читаемо) | для **разработчика** (однозначно) |
| Спец-метод | `__str__` | `__repr__` |
| Где видно | `print()` | в REPL, в контейнерах |

```python
s = "line\nbreak"
print(str(s))     # line
                  # break        — переносит строку
print(repr(s))    # 'line\nbreak' — показывает escape и кавычки

import datetime
d = datetime.date(2025, 1, 1)
print(str(d))     # 2025-01-01
print(repr(d))    # datetime.date(2025, 1, 1)
```

В REPL и при печати контейнеров используется `repr`:

```python
print(["a", "b"])   # ['a', 'b'] — элементы через repr (с кавычками)
```

## Встроенный объект str

```python
str()            # '' пустая
str(42)          # '42'
str(3.14)        # '3.14'
str([1, 2])      # '[1, 2]'
str(True)        # 'True'
```

## Базовые операции (как последовательность)

```python
s = "hello"
len(s)        # 5
s[0]          # 'h'
s[-1]         # 'o'
s[1:4]        # 'ell'
s[::-1]       # 'olleh'
"ab" + "cd"   # 'abcd'  конкатенация
"ab" * 3      # 'ababab'
"e" in s      # True
```

## Итоги

- `str` — **неизменяемая** Unicode-последовательность;
- литералы: `'...'`, `"..."`, тройные `"""..."""` (многострочные);
- `str()` — для человека (`__str__`), `repr()` — однозначно (`__repr__`);
- контейнеры и REPL показывают элементы через `repr`;
- «изменение» строки = новая строка.

# Билет 26: Строки как последовательности

## Строка — последовательность символов

Все неизменяющие операции последовательностей работают:

```python
s = "python"

len(s)        # 6
s[0]          # 'p'        индексация
s[-1]         # 'n'
s[1:4]        # 'yth'      срез
s[::-1]       # 'nohtyp'   разворот
s[::2]        # 'pto'      шаг
"py" + "thon" # 'python'   конкатенация
"ab" * 3      # 'ababab'   повторение
"th" in s     # True       вхождение подстроки
```

(!) Индексация даёт **строку из одного символа** (в Python нет отдельного типа char):

```python
print(type(s[0]))    # <class 'str'>
```

## Итерация по строке

```python
for ch in "abc":
    print(ch)         # a / b / c

chars = list("abc")   # ['a', 'b', 'c']
```

## in для подстрок

В отличие от списка, `in` проверяет **подстроку** целиком, а не один элемент:

```python
print("th" in "python")    # True  — подстрока
print("hto" in "python")   # False — не подряд
```

## Неизменяемость

Строка — последовательность, но **неизменяемая**: присваивание по индексу запрещено.

```python
s = "abc"
s[0] = "x"        # TypeError
s = "x" + s[1:]   # так — создаём новую строку "xbc"
```

## Полезные функции последовательностей

```python
s = "dcba"
sorted(s)         # ['a', 'b', 'c', 'd'] — СПИСОК символов
"".join(sorted(s))# 'abcd' — собрать обратно в строку
min("dcba")       # 'a'
max("dcba")       # 'd'
list(reversed(s)) # ['a', 'b', 'c', 'd']
```

## Конкатенация в цикле — антипаттерн

Каждое `+=` создаёт новую строку (O(n²) на цикле). Правильно — собрать список и `join`:

```python
# ❌ медленно
res = ""
for w in words:
    res += w

# ✅ быстро
res = "".join(words)
```

## Итоги

- строка — **неизменяемая** последовательность символов; `s[0]` — это `str` длины 1;
- работают: индексация, срезы, `+`, `*`, `in` (как подстрока!), итерация;
- `sorted(s)` → список символов, собрать через `"".join(...)`;
- сборка строки в цикле через `+=` — антипаттерн, нужен `join`.

## Задачи

### Задача 6.1. Длина: escape против r-строки

**Не запуская код**, верните из `escape_lengths()` кортеж
`(len("a\tb"), len(r"a\tb"))`.

### Задача 6.2. Замена символа в строке

Строки неизменяемы, поэтому `s[i] = ch` невозможно. Реализуйте `replace_at(s, i, ch)`,
которое возвращает **новую** строку, где символ на позиции `i` заменён на `ch`.

### Задача 6.3. Шифр Цезаря

`caesar(s, k)` — сдвинуть каждую строчную латинскую букву на `k` позиций по
кругу (после `z` снова `a`), остальные символы оставить как есть.

### Задача 6.4. Отсортирован ли по алфавиту

`is_sorted_words(words)` — вернуть `True`, если слова идут в неубывающем
лексикографическом порядке.

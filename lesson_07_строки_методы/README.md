# Урок 7. Строки: форматирование и методы

**Билеты:** 28–31

## Теория

Разбираемые билеты лежат в папке `теория/`:

- Билет 28. Форматирование строк
- Билет 29. Регистр и проверка состава строк
- Билет 30. Очистка, разбиение и объединение строк
- Билет 31. Поиск подстрок

## Задачи

### Задача 7.1. Процент с одним знаком

`pct(part, whole)` — вернуть строку вида `"42.9%"`: долю `part/whole` в процентах
с **одним** знаком после запятой.

### Задача 7.2. Надёжность пароля

`is_strong(pw)` — `True`, если длина не меньше 8 и есть хотя бы одна заглавная,
одна строчная и одна цифра.

### Задача 7.3. Разбор конфигурации

`parse_config(s)` — строка вида `"a = 1; b = hello world"`. Вернуть словарь
`{"a": "1", "b": "hello world"}`.

### Задача 7.4. Перекрывающиеся вхождения

`count_overlapping(s, sub)` — сколько раз `sub` встречается в `s`,
**считая перекрытия**. Например, в `"aaaa"` подстрока `"aa"` встречается
3 раза, а `str.count` вернёт лишь 2. Пустую `sub` считай за 0.

# Билет 28: Форматирование строк

## Способы форматирования

Современный Python — **f-строки** (3.6+) и метод `format()`. Старый `%` упоминают для полноты.

## f-строки (formatted string literals)

Префикс `f`; внутри `{}` — любое выражение:

```python
name = "Аня"
age = 25
print(f"{name}, {age} лет")          # Аня, 25 лет
print(f"{2 + 2}")                     # 4 — выражение
print(f"{name.upper()}")              # АНЯ — вызов метода

# Отладочный = (3.8+): печатает выражение и значение
x = 10
print(f"{x = }")                      # x = 10
```

## Метод format()

```python
"{}, {} лет".format(name, age)        # позиционные
"{0} и {1}, снова {0}".format("a", "b")  # по индексу → 'a и b, снова a'
"{n} лет".format(n=25)                # по имени
```

## Синтаксис формат-спецификации

После `:` внутри `{}` — мини-язык форматирования: `{value:[заполнение][выравнивание][знак][ширина][,][.точность][тип]}`

```python
pi = 3.14159
print(f"{pi:.2f}")        # 3.14    — 2 знака после точки
print(f"{pi:10.2f}")      # '      3.14' — ширина 10
print(f"{42:05d}")        # 00042   — дополнение нулями
print(f"{1000000:,}")     # 1,000,000 — разделитель тысяч
print(f"{0.25:.1%}")      # 25.0%   — проценты
print(f"{255:x}")         # ff      — hex
print(f"{255:b}")         # 11111111 — binary

# Выравнивание: < влево, > вправо, ^ по центру
print(f"{'hi':>8}")       # '      hi'
print(f"{'hi':<8}|")      # 'hi      |'
print(f"{'hi':^8}")       # '   hi   '
print(f"{'x':*^7}")       # '***x***' — заполнитель *
```

## Специальные методы выравнивания строк

Альтернатива формат-спекам — методы строк:

```python
"hi".center(7)        # '  hi   '
"hi".center(7, "*")   # '**hi***'
"hi".ljust(7)         # 'hi     '   — влево, добивка справа
"hi".rjust(7)         # '     hi'   — вправо, добивка слева
"hi".ljust(7, ".")    # 'hi.....'
"5".zfill(3)          # '005'       — добивка нулями
```

| Метод | Выравнивание |
|-------|--------------|
| `center(w)` | по центру |
| `ljust(w)` | влево |
| `rjust(w)` | вправо |
| `zfill(w)` | вправо нулями (для чисел) |

## Итоги

- f-строки `f"{выражение}"` — основной способ (выражения, методы, `{x=}`);
- `format()`: `{}`, `{0}`, `{name}`;
- спецификация после `:` — `.2f`, `05d`, `,`, `%`, `x`, выравнивание `< > ^`;
- методы выравнивания: `center`, `ljust`, `rjust`, `zfill`.

# Билет 30: Очистка, разбиение и объединение строк

## Замена: replace()

Заменяет все вхождения подстроки (или N первых):

```python
"a-b-c".replace("-", "+")      # 'a+b+c'
"a-b-c".replace("-", "+", 1)   # 'a+b-c' — только первое
"hello".replace("l", "")       # 'heo'  — удаление подстроки
```

## Очистка краёв: strip / lstrip / rstrip

Убирают символы **по краям** (по умолчанию пробельные):

```python
"  hi  ".strip()       # 'hi'   — с обоих краёв
"  hi  ".lstrip()      # 'hi  ' — слева
"  hi  ".rstrip()      # '  hi' — справа
"\n\t hi \n".strip()   # 'hi'

# С аргументом — убирает ЛЮБЫЕ из указанных символов (не подстроку!)
"xxhixx".strip("x")    # 'hi'
"www.site.com".strip("wcom.")   # 'site' — набор символов, не "www."!
"...a.b...".strip(".") # 'a.b'
```

(!) Аргумент `strip` — это **множество символов**, а не подстрока:

```python
"hello".strip("ho")    # 'ell' — убрало h слева и o справа
```

## Разбиение: split()

Разбивает строку в **список**:

```python
"a b c".split()           # ['a', 'b', 'c'] — по любым пробелам
"a,b,c".split(",")        # ['a', 'b', 'c'] — по разделителю
"a,,b".split(",")         # ['a', '', 'b']  — пустые сохраняются
"a-b-c".split("-", 1)     # ['a', 'b-c']    — максимум 1 разбиение
"a\nb\nc".splitlines()    # ['a', 'b', 'c'] — по строкам
"  ".split()              # []  — по пробелам пустые отбрасываются
"  ".split(" ")           # ['', '', '']    — с явным разделителем нет
"a b".rsplit(" ", 1)      # ['a', 'b'] — справа
```

(!) `split()` без аргумента и `split(" ")` ведут себя **по-разному** на множественных пробелах:

```python
"a   b".split()      # ['a', 'b']        — схлопывает пробелы
"a   b".split(" ")   # ['a', '', '', 'b'] — каждый пробел отдельно
```

## Объединение: join()

Обратное `split` — собирает итерируемое строк в одну строку. Вызывается **у разделителя**:

```python
"-".join(["a", "b", "c"])    # 'a-b-c'
"".join(["a", "b", "c"])     # 'abc'
", ".join(["x", "y"])        # 'x, y'
" ".join("abc")              # 'a b c' — итерируем по символам

# Элементы должны быть строками!
"-".join([1, 2, 3])          # TypeError
"-".join(str(n) for n in [1, 2, 3])   # '1-2-3'
```

## Идиома split→обработка→join

```python
s = "  hello,  world  "
parts = [p.strip() for p in s.split(",")]   # ['hello', 'world']
result = " | ".join(parts)                   # 'hello | world'
```

## Итоги

- `replace(old, new[, count])` — замена/удаление подстроки;
- `strip/lstrip/rstrip` убирают по краям; аргумент — **набор символов**, не подстрока;
- `split()` (по пробелам, схлопывает) vs `split(sep)` (по разделителю, пустые сохраняет);
- `sep.join(iterable)` — объединение; элементы должны быть `str`.

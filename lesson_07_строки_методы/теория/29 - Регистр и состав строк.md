# Билет 29: Регистр и проверка состава строк

## Манипуляция регистром

Все возвращают **новую** строку (строка неизменяема):

```python
"Hello".lower()      # 'hello'
"Hello".upper()      # 'HELLO'
"hello world".title()# 'Hello World' — каждое слово с заглавной
"Hello".capitalize() # 'Hello' — только первая буква
"Hello".swapcase()   # 'hELLO' — инверсия регистра
"Straße".casefold()  # для надёжного регистронезависимого сравнения
```

## Проверка регистра

Возвращают `bool`:

```python
"hello".islower()    # True
"Hello".islower()    # False
"HELLO".isupper()    # True
"Hello".istitle()    # True — каждое слово с заглавной?
```

(!) Строки без букв дают `False`:

```python
"123".islower()      # False — нет букв в нижнем регистре
"".isupper()         # False
```

## Проверка состава строки

| Метод | True, если строка непуста и все символы… |
|-------|--------------------------------------------|
| `isalpha()` | буквы |
| `isdigit()` | цифры |
| `isalnum()` | буквы или цифры |
| `isspace()` | пробельные |
| `isnumeric()` | числовые (шире digit) |
| `isidentifier()` | валидный идентификатор |

```python
"abc".isalpha()      # True
"abc1".isalpha()     # False
"123".isdigit()      # True
"12.5".isdigit()     # False — точка не цифра
"abc123".isalnum()   # True
"  \t\n".isspace()   # True
"".isalpha()         # False — пустая строка всегда False!
```

(!) На **пустой** строке все `is...` дают `False`.

## Модуль string и наборы символов

В стандартной библиотеке `string` есть готовые наборы:

```python
import string

string.ascii_lowercase   # 'abcdefghijklmnopqrstuvwxyz'
string.ascii_uppercase   # 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
string.ascii_letters     # обе вместе
string.digits            # '0123456789'
string.punctuation       # '!"#$%&...'
string.whitespace        # ' \t\n\r\x0b\x0c'
string.printable         # всё печатаемое
```

Применение — проверки и генерация:

```python
import string, random

# Случайный пароль
chars = string.ascii_letters + string.digits
pwd = "".join(random.choice(chars) for _ in range(8))

# Проверка: только буквы и пробелы
def only_letters_spaces(s):
    allowed = set(string.ascii_letters + " ")
    return all(c in allowed for c in s)
```

## Итоги

- регистр: `lower upper title capitalize swapcase casefold` → новая строка;
- проверка регистра: `islower isupper istitle`;
- состав: `isalpha isdigit isalnum isspace` — все `False` на **пустой** строке;
- модуль `string`: `ascii_letters`, `digits`, `punctuation`, `whitespace`.

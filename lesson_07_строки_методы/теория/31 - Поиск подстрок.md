# Билет 31: Поиск подстрок

## Проверка начала/конца: startswith / endswith

Возвращают `bool`:

```python
"report.pdf".endswith(".pdf")     # True
"report.pdf".startswith("rep")    # True

# Можно передать КОРТЕЖ вариантов
"image.png".endswith((".png", ".jpg"))   # True
"http://x".startswith(("http://", "https://"))   # True
```

(!) Передавать список нельзя — только кортеж:

```python
"x".startswith(["a", "b"])   # TypeError
```

## Поиск позиции: find / rfind

Возвращают **индекс** первого/последнего вхождения, или **-1**, если не найдено:

```python
s = "abcabc"
s.find("b")     # 1  — первое вхождение
s.rfind("b")    # 4  — последнее (поиск справа)
s.find("x")     # -1 — НЕ найдено (не исключение!)
s.find("b", 2)  # 4  — поиск с позиции 2
s.find("b", 2, 4)  # -1 — в срезе [2:4] нет 'b'
```

## Поиск позиции: index / rindex

Как `find`/`rfind`, но при отсутствии бросают **ValueError**:

```python
s = "abcabc"
s.index("b")    # 1
s.rindex("b")   # 4
s.index("x")    # ValueError: substring not found
```

| Метод | Не найдено |
|-------|------------|
| `find` / `rfind` | возвращает `-1` |
| `index` / `rindex` | `ValueError` |

## Главное различие find vs index

`find` безопасен (−1), `index` — строгий (исключение). Выбор зависит от того, ожидаемо ли отсутствие:

```python
pos = s.find("x")
if pos != -1:               # безопасная проверка
    ...

# Если -1 может быть валидным индексом, find опасен:
if "x" in s:                # сначала проверить вхождение
    pos = s.index("x")
```

## Подсчёт: count

```python
"abcabc".count("a")    # 2
"abcabc".count("bc")   # 2
"aaa".count("aa")      # 1 — без перекрытия!
"abcabc".count("a", 1) # 1 — с позиции 1
```

(!) `count` не считает **перекрывающиеся** вхождения (`"aaa".count("aa") == 1`, не 2).

## Проверка вхождения: in

Самый простой способ узнать «есть ли подстрока» (без позиции):

```python
"bc" in "abcabc"    # True
"xy" in "abcabc"    # False
```

## Итоги

- `startswith`/`endswith` → bool, принимают **кортеж** вариантов;
- `find`/`rfind` → индекс или **-1**; `index`/`rindex` → индекс или **ValueError**;
- выбор `find` vs `index`: ожидаемо ли отсутствие;
- `count` — без перекрытий (`"aaa".count("aa")==1`);
- просто «есть ли» → оператор `in`.

# Билет 11: Числа с плавающей точкой (float)

## Тип float

`float` — вещественные числа по стандарту **IEEE 754 double** (64 бита). Свойства:

1. **Неизменяемость (immutable)** — как у int.
2. **Ограниченный диапазон и точность** — в отличие от int! Примерно ±1.8·10³⁰⁸, ~15–17 значащих десятичных цифр.

```python
import sys
print(sys.float_info.max)   # 1.7976931348623157e+308
print(1e308 * 10)           # inf — переполнение!
```

## Валидные литералы float

```python
3.14
0.5
.5             # ведущий ноль можно опустить
5.             # дробную часть можно опустить
1e3            # 1000.0 — экспоненциальная запись
2.5e-4         # 0.00025
1_000.0        # подчёркивания тоже работают
```

## Специальные значения: inf, -inf, nan

IEEE 754 определяет особые значения. Получить можно через `float()` или модуль `math`:

```python
import math

pos_inf = float("inf")    # или math.inf
neg_inf = float("-inf")   # или -math.inf
not_a_num = float("nan")  # или math.nan
```

**Смысл:**
- `inf` / `-inf` — результат переполнения или, например, `1/0.0`-подобных операций (через `math`), «больше/меньше любого числа»;
- `nan` (Not a Number) — «неопределённый результат», например `inf - inf` или `0 * inf`.

```python
print(math.inf > 10**100)   # True
print(math.inf - math.inf)  # nan
```

## Главная особенность nan

`nan` **не равен ничему, включая сам себя**:

```python
n = float("nan")
print(n == n)            # False !!!
print(n != n)            # True
```

Поэтому проверять на nan надо специальной функцией:

```python
print(math.isnan(n))     # True — правильный способ
print(math.isinf(math.inf))  # True
```

## Встроенный объект float

```python
float()          # 0.0
float(5)         # 5.0
float("3.14")    # 3.14
float("inf")     # inf
float("1e-3")    # 0.001
(3.0).is_integer()   # True — дробная часть нулевая
```

## Точность

`float` хранит число приближённо в двоичной системе — отсюда классическое:

```python
print(0.1 + 0.2)            # 0.30000000000000004
print(0.1 + 0.2 == 0.3)     # False
```

## Итоги

- `float` — IEEE 754 double: **неизменяем**, но **ограничен** по диапазону/точности (в отличие от int);
- спец-значения: `inf`, `-inf`, `nan`;
- `nan != nan` → проверять `math.isnan()`;
- `0.1 + 0.2 != 0.3` — следствие двоичного представления.

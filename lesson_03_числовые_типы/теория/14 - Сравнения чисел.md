# Билет 14: Операции сравнения числовых типов

## Операторы сравнения

```python
3 == 3     # True   равно (по значению)
3 != 4     # True   не равно
3 < 4      # True
3 <= 3     # True
5 > 2      # True
5 >= 6     # False
```

Результат сравнения — всегда `bool`. Числа разных типов сравниваются по значению:

```python
print(3 == 3.0)        # True  (int и float)
print(1 == True)       # True  (bool — подкласс int)
print(2 < 3.5 < 4)     # True
```

## Цепочки сравнений

Python позволяет **цепочки** — то, чего нет в C/Java. `a < b < c` означает `(a < b) and (b < c)`, причём `b` вычисляется **один раз**:

```python
x = 5
print(1 < x < 10)        # True  ≡ (1 < x) and (x < 10)
print(1 < x < 3)         # False
print(0 <= x <= 100)     # True

# Можно любой длины и с разными операторами
print(1 < 2 == 2 < 3)    # True
```

## Особенности сравнения float

Из-за приближённого хранения **прямое `==` для float ненадёжно**:

```python
print(0.1 + 0.2 == 0.3)   # False!
print(0.1 + 0.2)          # 0.30000000000000004
```

Сравнивать float на равенство надо **с допуском**.

## Функция math.isclose()

`math.isclose(a, b)` сравнивает с относительным и абсолютным допуском:

```python
import math

print(math.isclose(0.1 + 0.2, 0.3))   # True ✅

# Параметры допуска
print(math.isclose(1000.0, 1000.1, rel_tol=1e-3))   # True
print(math.isclose(0.0, 1e-10, abs_tol=1e-9))        # True
```

Сигнатура: `isclose(a, b, *, rel_tol=1e-09, abs_tol=0.0)`.
- `rel_tol` — относительный допуск (доля от большего по модулю);
- `abs_tol` — абсолютный (важен при сравнении с нулём, т.к. относительный около 0 не работает).

```python
# Вблизи нуля нужен abs_tol:
print(math.isclose(0.0, 1e-12))               # False (rel_tol бесполезен у 0)
print(math.isclose(0.0, 1e-12, abs_tol=1e-9)) # True
```

## nan в сравнениях

`nan` несравним — любое сравнение с ним даёт `False` (кроме `!=`):

```python
n = float("nan")
print(n < 1, n > 1, n == n)   # False False False
print(n != n)                 # True
```

## Итоги

- сравнения возвращают `bool`; `3 == 3.0` → True;
- **цепочки**: `1 < x < 10` ≡ `(1<x) and (x<10)`, среднее вычисляется один раз;
- float **нельзя** сравнивать через `==` → `math.isclose`;
- у нуля нужен `abs_tol`; `rel_tol` по умолчанию `1e-9`;
- `nan` несравним: все сравнения `False`, кроме `!=`.

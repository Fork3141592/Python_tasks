# Билет 13: Арифметические операции

## Базовые операции

```python
7 + 2      # 9   сложение
7 - 2      # 5   вычитание
7 * 2      # 14  умножение
7 ** 2     # 49  возведение в степень
7 / 2      # 3.5 деление — ВСЕГДА float
7 // 2     # 3   целочисленное деление (floor)
7 % 2      # 1   остаток (modulo)
```

## Особенность // и % в Python

`/` всегда возвращает **float**, даже если делится нацело:

```python
print(6 / 2)       # 3.0  (не 3!)
print(type(6 / 2)) # <class 'float'>
```

`//` — деление **с округлением вниз (floor)**, а не усечением к нулю. Это важно для отрицательных:

```python
print(7 // 2)      # 3
print(-7 // 2)     # -4   (floor: -3.5 округляется ВНИЗ до -4, не -3!)
print(7 // -2)     # -4
```

`%` в Python берёт знак **делителя** (а не делимого, как в C):

```python
print(-7 % 2)      # 1    (в C было бы -1)
print(7 % -2)      # -1
```

Связь сохраняется: `a == (a // b) * b + (a % b)`:

```python
print((-7 // 2) * 2 + (-7 % 2))   # -7 ✓
print(divmod(-7, 2))               # (-4, 1)
```

## Операции составного присваивания

Сокращение «применить операцию и переприсвоить»:

```python
x = 10
x += 3     # x = x + 3  → 13
x -= 1     # 12
x *= 2     # 24
x /= 5     # 4.8  (стало float!)
x //= 2    # 2.0
x **= 3    # 8.0
x %= 5     # 3.0
```

Для **неизменяемых** типов `x += 1` создаёт новый объект и переприсваивает `x`. Для **изменяемых** (список) — может менять «на месте»:

```python
lst = [1, 2]
print(id(lst))
lst += [3]          # для списка == lst.extend([3]) — тот же объект!
print(id(lst))      # тот же id
```

## Приоритет операций

`**` (правоассоциативно) > унарный `-` > `* / // %` > `+ -`:

```python
print(2 ** 3 ** 2)   # 512  (= 2 ** 9, право-ассоциативно)
print(-2 ** 2)       # -4   (= -(2**2), не (-2)**2)
print(2 + 3 * 4)     # 14
```

## Итоги

- `/` всегда float; `//` — floor (вниз), не усечение;
- `-7 // 2 == -4`, `-7 % 2 == 1` (знак остатка = знак делителя);
- `a == (a//b)*b + (a%b)`, `divmod(a,b)`;
- составное присваивание: для mutable (`+=` к списку) меняет на месте;
- `**` право-ассоциативно; `-2 ** 2 == -4`.

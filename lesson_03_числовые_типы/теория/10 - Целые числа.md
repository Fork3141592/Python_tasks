# Билет 10: Целые числа (int)

## Тип int

`int` — целочисленный тип. Два ключевых свойства в Python:

1. **Неизменяемость (immutable).** Целое число нельзя изменить «на месте» — арифметика создаёт новый объект.
2. **Неограниченный диапазон.** В отличие от C (`int32`, `int64`), Python-`int` имеет **произвольную точность** — ограничен только памятью.

```python
big = 2 ** 1000          # 302-значное число — без переполнения
print(big)
print(type(big))         # <class 'int'>
```

### Неизменяемость на практике

```python
x = 5
print(id(x))
x += 1                   # создаётся НОВЫЙ объект 6
print(id(x))             # другой id — x не изменён, а перепривязан
```

## Валидные целочисленные литералы

```python
42             # десятичный
0b1010         # двоичный (0b) = 10
0o17           # восьмеричный (0o) = 15
0xFF           # шестнадцатеричный (0x) = 255
1_000_000      # подчёркивания-разделители для читаемости = 1000000
-7             # знак — это унарный оператор, не часть литерала
```

Ошибки:

```python
007            # ❌ SyntaxError (ведущие нули в десятичном запрещены)
1_            # ❌ подчёркивание не на краю/не между цифрами
```

## Встроенный объект int

`int` — это и тип, и **конструктор/функция приведения**:

```python
int()           # 0
int(3.9)        # 3   — усечение к нулю (не округление!)
int("42")       # 42
int("1010", 2)  # 10  — строка в системе счисления 2
int("FF", 16)   # 255
int(True)       # 1
```

```python
print(int(-3.9))   # -3 (усечение к нулю, не -4)
```

## Полезные операции

```python
abs(-5)          # 5
pow(2, 10)       # 1024  (== 2 ** 10)
divmod(17, 5)    # (3, 2) — частное и остаток
(255).bit_length()  # 8 — сколько бит нужно
```

## Итоги

- `int` **неизменяем** и имеет **произвольную точность** (нет переполнения);
- литералы: `0b`, `0o`, `0x`, `_` как разделитель; ведущие нули запрещены;
- `int(x)` усекает дробь **к нулю** (`int(-3.9) == -3`);
- `int(s, base)` — разбор строки в заданной системе счисления.

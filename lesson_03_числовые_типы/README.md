# Урок 3. Числовые типы и операции

**Билеты:** 10–15

## Теория

Разбираемые билеты лежат в папке `теория/`:

- Билет 10. Целые числа (int)
- Билет 11. Числа с плавающей точкой (float)
- Билет 12. Логический тип, синглтоны True/False/None
- Билет 13. Арифметические операции
- Билет 14. Операции сравнения числовых типов
- Билет 15. Логические операторы not, and, or

## Задачи

### Задача 3.1. Сравнение float

`0.1 + 0.2 == 0.3` даёт `False` — это не баг, а двоичное представление.
Реализуй `safe_equal(a, b)`, которое считает два float равными, если они близки.

### Задача 3.2. Часы по кругу

`clock_add(h, delta)` — прибавить `delta` часов к `h` на 24-часовом циферблате и
вернуть результат в диапазоне `0..23`. Работать должно и для отрицательного
`delta`.

### Задача 3.3. Большинство голосов

`majority(votes)` — список из `True/False`. Вернуть `True`, если голосов «за»
**строго больше** половины. Ничья — не большинство.

### Задача 3.4. coalesce: первый не-None

`coalesce(*args)` — вернуть первый аргумент, который **не равен `None`**; если
таких нет — `None`.

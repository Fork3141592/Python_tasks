# Билет 8: Циклы while, for, break/continue, else

## Цикл while

Повторяет тело, **пока условие истинно**:

```python
i = 0
while i < 5:
    print(i)
    i += 1            # важно менять условие — иначе вечный цикл
```

## Цикл for

Перебирает элементы **итерируемого объекта** (список, строка, range, словарь, генератор…):

```python
for ch in "abc":
    print(ch)         # a, b, c

for i in range(3):
    print(i)          # 0, 1, 2

for k, v in {"a": 1}.items():
    print(k, v)
```

В Python `for` — это **foreach**: нет «цикла со счётчиком» в стиле C, есть перебор элементов. Счётчик получают через `range()` или `enumerate()`.

```python
for idx, val in enumerate(["a", "b"]):
    print(idx, val)   # 0 a / 1 b
```

## break и continue

```python
for x in range(10):
    if x == 3:
        continue      # пропустить остаток ИТЕРАЦИИ, перейти к следующей
    if x == 6:
        break         # выйти из ЦИКЛА полностью
    print(x)          # 0 1 2 4 5
```

| Инструкция | Действие |
|-----------|----------|
| `continue` | перейти к следующей итерации |
| `break` | немедленно выйти из цикла |

## Инструкция else в циклах

Особенность Python: у `for` и `while` может быть блок **`else`**. Он выполняется, **если цикл завершился нормально** (не был прерван `break`).

```python
for x in [1, 2, 3]:
    if x == 5:
        break
else:
    print("5 не найдено")   # выполнится: break не сработал

for x in [1, 2, 5]:
    if x == 5:
        break
else:
    print("не выводится")   # пропущено: был break
```

Мнемоника: `else` = «**no-break**» (если не было `break`). Удобно для поиска:

```python
def find(seq, target):
    for item in seq:
        if item == target:
            print("найдено")
            break
    else:
        print("не найдено")   # цикл прошёл до конца без break
```

## Итоги

- `while` — по условию, `for` — по элементам (foreach, не счётчик);
- `continue` → следующая итерация; `break` → выход из цикла;
- `else` у цикла = «цикл не прервали `break`» (no-break);
- счётчик в `for` — через `range`/`enumerate`.

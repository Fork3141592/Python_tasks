# Билет 7: Инструкции и ветвление

## Инструкция (statement)

**Инструкция** — единица выполнения программы. Делятся на простые и составные.

## Простые инструкции

Умещаются в одну логическую строку, не содержат вложенных блоков:

```python
x = 5            # присваивание
print(x)         # вызов
del x            # удаление имени
pass             # пустая операция (заглушка)
return value     # возврат из функции
raise ValueError # возбуждение исключения
import math      # импорт
assert x > 0     # проверка
break / continue # управление циклом
global x         # объявление
```

## Составные инструкции

Содержат **заголовок** (заканчивается `:`) и вложенный **блок** (тело с отступом):

```python
if ...:          # ветвление
while ...:       # цикл с условием
for ...:         # цикл по итерируемому
try ...:         # обработка исключений
with ...:        # менеджер контекста
def ...:         # определение функции
class ...:       # определение класса
match ...:       # сопоставление с образцом
```

Структура:

```python
заголовок:
    инструкция_1     # тело — блок с одинаковым отступом
    инструкция_2
```

## Ветвление в Python

Условный выбор — конструкция `if / elif / else`:

```python
if condition1:
    ...                # выполнится, если condition1 истинно
elif condition2:
    ...                # иначе если condition2
elif condition3:
    ...
else:
    ...                # если ни одно условие не истинно
```

- `elif` и `else` **необязательны**;
- `elif` может быть сколько угодно;
- проверяется **истинностное значение** выражения (см. билет 12), не обязательно `bool`.

```python
x = 0
if x:                 # 0 → False
    print("truthy")
else:
    print("falsy")    # сюда
```

## Тернарное выражение

Компактная форма выбора **значения** (это выражение, а не инструкция):

```python
sign = "pos" if x > 0 else "non-pos"
```

## В Python нет switch (есть match)

Классического `switch` нет; для множественного ветвления по образцу — `match-case` (билет 9).

## Итоги

- простые инструкции — в одну логическую строку; составные — заголовок + блок;
- ветвление: `if / elif* / else?`;
- условие проверяется на **истинностность** (truthy/falsy), не только `bool`;
- тернарник: `A if cond else B`;
- `switch` отсутствует — вместо него `match-case`.

## Задачи

### Задача 2.1. Классификация точки

Через `match-case` определите положение точки `(x, y)`:
`(0,0)`→`"origin"`, точка на оси Y `(0, y)`→`"on-y"`, на оси X `(x, 0)`→`"on-x"`,
иначе `"general"`. Порядок шаблонов важен — начните с самого частного.

### Задача 2.2. Длина последовательности Коллатца

Для натурального `n` повторяем: если чётное — делим на 2, если нечётное —
`3n+1`. Верните число шагов до достижения 1. `collatz_steps(1) == 0`.

### Задача 2.3. Первый повтор

Верните первый элемент, который встречается в последовательности не в первый раз;
если повторов нет — `None`.

### Задача 2.4. Интерпретатор команд

Реализуйте `run(command, pos)`, где `pos = (x, y)`. Команда — список:
- `["move", d, n]`, где `d` из `"up"/"down"/"left"/"right"`, а `n` — целое число
  шагов: сдвинуть точку. `up`→`+y`, `down`→`-y`, `right`→`+x`, `left`→`-x`.
- `["reset"]` → `(0, 0)`.
- всё остальное (в т.ч. если `n` не `int`) → вернуть `pos` без изменений.

# Билет 6: Токены

## Что такое токен

При разборе исходного кода интерпретатор сначала разбивает текст на **токены** (лексемы) — минимальные значимые единицы. Категории токенов в Python:

## Разделители (delimiters)

Символы, разделяющие/структурирующие код:

```python
( ) [ ] { }        # скобки
, : . ; @ =        # запятая, двоеточие, точка, ; , декоратор, =
+= -= *= /= //= **= %= &= |= ^= >>= <<=   # составные присваивания
-> ...             # стрелка аннотации, ellipsis
```

## Литералы (literals)

Записи значений «как есть» в коде:

```python
42            # целочисленный литерал
3.14          # литерал float
1 + 2j        # комплексный
"text"        # строковый
b"bytes"      # байтовый
True False None   # литералы-константы (формально keyword)
[1, 2]        # списковый литерал
(1, 2)        # кортежный
{1, 2}        # множественный
{"a": 1}      # словарный
```

## Операторы (operators)

Символы операций:

```python
+ - * / // % **      # арифметика
== != < > <= >=      # сравнение
and or not           # логические (это и операторы, и keyword)
& | ^ ~ << >>        # побитовые
is  in               # идентичность, принадлежность
```

## Ключевые слова (keywords)

Зарезервированные имена, которые **нельзя** использовать как идентификаторы. Узнать список:

```python
import keyword
print(keyword.kwlist)
# ['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await',
#  'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except',
#  'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is',
#  'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try',
#  'while', 'with', 'yield']
```

## Мягкие ключевые слова (soft keywords)

Особый класс: они **ключевые только в определённом контексте**, а в остальных местах могут быть обычными именами. Появились с Python 3.10:

```python
import keyword
print(keyword.softkwlist)   # ['_', 'case', 'match', 'type']
```

```python
match = 5            # OK! 'match' здесь обычная переменная
match command:       # а здесь 'match' — ключевое слово
    case "go":
        ...
```

> В отличие от обычных keyword, `match`, `case`, `type`, `_` не запрещены как имена.

## Итоги

| Категория | Примеры |
|-----------|---------|
| разделители | `( ) , : . ; @` |
| литералы | `42`, `"text"`, `[1,2]` |
| операторы | `+ == and is in` |
| ключевые слова | `if def return class` |
| мягкие ключевые | `match case type _` |

- 5 категорий токенов;
- `keyword.kwlist` и `keyword.softkwlist`;
- мягкие ключевые слова (`match`, `case`, `type`, `_`) — keyword только в контексте.

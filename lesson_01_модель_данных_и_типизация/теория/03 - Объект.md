# Билет 3: Объект — идентичность, тип, значение

## Всё в Python — объект

Числа, строки, списки, функции, классы, модули — **всё** является объектами. У каждого объекта есть три неотъемлемые характеристики.

## Три характеристики объекта

| Характеристика | Что это | Чем узнать | Меняется? |
|----------------|---------|------------|-----------|
| **Идентичность** (identity) | «кто это», уникальный признак объекта за время его жизни | `id(obj)` | никогда |
| **Тип** (type) | класс объекта, определяет поведение | `type(obj)` | никогда |
| **Значение** (value) | данные, которые объект хранит | сам объект | зависит от изменяемости |

```python
x = [1, 2, 3]
print(id(x))     # идентичность, напр. 140234...
print(type(x))   # <class 'list'> — тип
print(x)         # [1, 2, 3] — значение
```

**Идентичность и тип неизменны** в течение жизни объекта. Значение может меняться — но только у изменяемых объектов.

## Изменяемые и неизменяемые типы

- **Неизменяемые (immutable):** `int`, `float`, `bool`, `str`, `bytes`, `tuple`, `frozenset`, `None`. Значение нельзя изменить после создания — любая «модификация» создаёт **новый** объект.
- **Изменяемые (mutable):** `list`, `dict`, `set`, bytearray, экземпляры обычных классов. Значение можно менять «на месте», `id` при этом сохраняется.

```python
# Неизменяемый: id меняется при "изменении"
s = "abc"
print(id(s))
s += "d"          # создаётся НОВАЯ строка
print(id(s))      # другой id

# Изменяемый: id сохраняется
lst = [1, 2, 3]
print(id(lst))
lst.append(4)     # меняем тот же объект
print(id(lst))    # тот же id
```

## Почему это важно

Изменяемость определяет поведение при присваивании и передаче в функции:

```python
a = [1, 2]
b = a            # b ссылается на ТОТ ЖЕ объект
b.append(3)
print(a)         # [1, 2, 3] — изменился и a!

x = 1
y = x
y += 1           # создаётся новый int, x не трогается
print(x)         # 1
```

## Итоги

- у каждого объекта: идентичность + тип + значение;
- идентичность и тип **неизменны**; значение — только у mutable;
- immutable: `int; float; bool; str; tuple; frozenset; bytes; None`;
- mutable: `list; dict; set; bytearray`;
- «изменение» неизменяемого объекта = создание нового объекта.

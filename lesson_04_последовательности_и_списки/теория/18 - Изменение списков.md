# Билет 18: Изменение списков

## Чтение элементов

```python
s = [10, 20, 30]
print(s[1])      # 20
print(s[1:3])    # [20, 30]
```

## Изменение элементов

Список изменяем — присваивание по индексу/срезу меняет объект «на месте» (id сохраняется):

```python
s = [10, 20, 30]
s[0] = 99           # [99, 20, 30]
s[1:3] = [0, 0, 0]  # [99, 0, 0, 0] — срез можно заменить срезом другой длины
```

## Добавление элементов

```python
s = [1, 2]
s.append(3)        # [1, 2, 3]      — один элемент в конец
s.extend([4, 5])   # [1,2,3,4,5]    — добавить элементы итерируемого
s += [6]           # как extend
s.insert(0, 0)     # [0,1,2,3,4,5,6] — вставить по индексу
```

Различие `append` и `extend`:

```python
a = [1, 2]; a.append([3, 4]); print(a)   # [1, 2, [3, 4]]  — вложенный список!
b = [1, 2]; b.extend([3, 4]); print(b)   # [1, 2, 3, 4]    — поэлементно
```

## Удаление элементов

```python
s = [10, 20, 30, 20]
s.remove(20)     # удаляет ПЕРВОЕ вхождение значения → [10, 30, 20]
val = s.pop()    # удаляет и ВОЗВРАЩАЕТ последний → val=20, s=[10, 30]
val = s.pop(0)   # по индексу → val=10, s=[30]
del s[0]         # удаление по индексу/срезу
s.clear()        # очистить полностью → []
```

| Метод | Что удаляет | Возвращает |
|-------|-------------|------------|
| `remove(x)` | первое значение `x` | `None` (ValueError если нет) |
| `pop(i)` | по индексу (по умолч. последний) | удалённый элемент |
| `del s[i]` | по индексу/срезу | — |
| `clear()` | всё | — |

## Прочие изменяющие методы

```python
s = [3, 1, 2]
s.sort()         # сортировка на месте → [1, 2, 3]; возвращает None!
s.reverse()      # разворот на месте → [3, 2, 1]
```

## Методы изменения возвращают None

Типичная ошибка:

```python
s = [3, 1, 2]
s = s.sort()     # ❌ s стало None! sort() меняет на месте и вернул None
```

Правильно: `s.sort()` без присваивания, либо `sorted(s)` для нового списка.

## Итоги

- изменение по индексу/срезу — на месте, id сохраняется;
- `append` — один элемент, `extend`/`+=` — поэлементно;
- удаление: `remove(value)`, `pop(index)→value`, `del`, `clear()`;
- `sort()`/`reverse()`/`append()` возвращают **None** (меняют на месте);
- не писать `s = s.sort()`.

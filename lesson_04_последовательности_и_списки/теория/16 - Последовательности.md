# Билет 16: Последовательности

## Что такое последовательность

**Последовательность (sequence)** — это упорядоченная коллекция элементов, доступных по **целочисленному индексу**. К последовательностям относятся: `list`, `tuple`, `str`, `range`, `bytes`.

## Основные характеристики

### 1. Упорядоченность и индексация

Элементы имеют позиции `0, 1, 2, …`; доступ по индексу:

```python
seq = [10, 20, 30]
print(seq[0])    # 10  — первый
print(seq[2])    # 30
print(seq[-1])   # 30  — отрицательный индекс: с конца
print(seq[-2])   # 20
```

### 2. Длина

```python
print(len([1, 2, 3]))   # 3
print(len("abc"))        # 3
```

### 3. Срезы (slicing)

`seq[start:stop:step]` — подпоследовательность:

```python
s = [0, 1, 2, 3, 4, 5]
print(s[1:4])     # [1, 2, 3]   — [start:stop), stop не включается
print(s[:3])      # [0, 1, 2]
print(s[3:])      # [3, 4, 5]
print(s[::2])     # [0, 2, 4]   — каждый второй
print(s[::-1])    # [5,4,3,2,1,0] — разворот
```

### 4. Конкатенация и повторение

```python
print([1, 2] + [3, 4])    # [1, 2, 3, 4]
print([0] * 3)            # [0, 0, 0]
print("ab" * 2)           # 'abab'
```

### 5. Проверка вхождения (in)

```python
print(3 in [1, 2, 3])     # True
print("x" in "abc")       # False
```

### 6. Итерируемость

По любой последовательности можно пройти `for`:

```python
for x in (10, 20, 30):
    print(x)
```

### 7. Общие функции и методы

```python
seq = [3, 1, 2]
len(seq)          # 3
min(seq), max(seq)# 1, 3
sum(seq)          # 6
seq.index(1)      # 1 — позиция значения
seq.count(3)      # 1 — сколько раз встречается
list(reversed(seq))
sorted(seq)       # [1, 2, 3]
```

## Изменяемые vs неизменяемые последовательности

- **изменяемые:** `list`
- **неизменяемые:** `tuple`, `str`, `range`, `bytes`

У неизменяемых нет методов модификации (`append`, присваивание по индексу и т.п.).

## Итоги

- последовательности: `list tuple str range bytes`;
- общие свойства: индексация (в т.ч. отрицательная), `len`, срезы, `+`, `*`, `in`, итерация;
- срез `[start:stop:step]`, `stop` не включается, `[::-1]` — разворот;
- `list` изменяем, остальные — нет.

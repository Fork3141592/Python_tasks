# Билет 17: Списки (list)

## Что такое список

`list` — **изменяемая упорядоченная** последовательность произвольных объектов (можно смешивать типы).

```python
nums = [1, 2, 3]
mixed = [1, "two", 3.0, [4, 5]]   # разные типы — норм
empty = []
```

## Списковые литералы

```python
[]                  # пустой
[1, 2, 3]           # из элементов
list("abc")         # ['a', 'b', 'c'] — из итерируемого
list(range(3))      # [0, 1, 2]
```

## Списковые включения (list comprehensions)

Компактный способ построить список из итерируемого:

```python
squares = [x**2 for x in range(5)]          # [0, 1, 4, 9, 16]

# С условием (фильтр)
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# С тернарником (преобразование)
signs = [("+" if x > 0 else "-") for x in [-1, 2, -3]]  # ['-', '+', '-']

# Вложенные циклы
pairs = [(i, j) for i in range(2) for j in range(2)]
# [(0,0), (0,1), (1,0), (1,1)]
```

Включения читаемее и обычно быстрее, чем `for` + `append`.

## Встроенный объект list

`list` — тип и конструктор:

```python
list()          # []
list((1, 2))    # [1, 2]
list("hi")      # ['h', 'i']
```

## Хранение в памяти

Список хранит **ссылки** на объекты, а не сами объекты «в ряд». Поэтому он гетерогенный, а элементы могут быть где угодно в памяти. Сам список — массив указателей с запасом ёмкости (для быстрого `append`).

```python
a = [1, 2, 3]
b = [a, a]       # b хранит две ССЫЛКИ на один список a
a.append(99)
print(b)         # [[1, 2, 3, 99], [1, 2, 3, 99]] — обе «видят» изменение
```

## Индексация

```python
s = [10, 20, 30, 40]
s[0]     # 10
s[-1]    # 40
s[10]    # IndexError: list index out of range
```

## Срезы списков

```python
s = [0, 1, 2, 3, 4]
s[1:4]      # [1, 2, 3]
s[:2]       # [0, 1]
s[::-1]     # [4, 3, 2, 1, 0]
s[1:4:2]    # [1, 3]

# Срез создаёт НОВЫЙ список (поверхностная копия)
copy = s[:]
print(copy is s)   # False
```

## Итоги

- `list` — изменяемый, упорядоченный, гетерогенный;
- включение: `[expr for x in it if cond]` — читаемо и быстро;
- список хранит **ссылки**; `[a, a]` — две ссылки на один объект;
- индекс вне диапазона → `IndexError`;
- срез возвращает **новый** список (поверхностная копия), `s[:]`.

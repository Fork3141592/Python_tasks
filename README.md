# Python — экспериментальный курс по билетам МФТИ ФАЛТ

## Как начать

1. **Форкните** этот репозиторий к себе на аккаунт GitHub к своим репозиториям (кнопка *Fork* справа сверху).
2. Создайте отдельную папку и клонируйте свой форк (путь можно посмотреть зайдя на свой репозиторий):
   ```bash
   git clone https://github.com/<твой-логин>/<репозиторий>.git
   cd <репозиторий>
   ```
3. Поставьте зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Как решать

Каждый урок лежит в папке `lesson_NN_.../`:

```
README.md        — условия задач
теория/          — исходные билеты
solution.py      — ФАЙЛ, КОТОРЫЙ ВЫ РЕДАКТИРУЕТЕ (в нём заготовки функций)
```

Порядок работы:

1. Откройте `solution.py` в нужном уроке и допишите тела функций.
2. Запусти проверку из папки урока:
   ```bash
   cd lesson_01_модель_данных_и_типизация
   pytest test_lesson.py -v
   ```
3. Зелёные тесты — задача решена. Красные — читайте сообщение об ошибке.

Проверить одну задачу:
```bash
pytest test_lesson.py -k task_2
```

## Автопроверка на GitHub

При каждом `push` в ваш форк GitHub Actions сам прогоняет все тесты
(вкладка **Actions**). Когда все уроки решены — сборка станет зелёной ✅.

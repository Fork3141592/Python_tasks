# Билет 22: Кортежи (tuple)

## Что такое кортеж

`tuple` — **неизменяемая** упорядоченная последовательность. Как список, но менять нельзя.

```python
t = (1, 2, 3)
t[0] = 99      # TypeError: 'tuple' object does not support item assignment
```

## Кортежные литералы

Кортеж определяет **запятая**, а не скобки:

```python
t = (1, 2, 3)
t = 1, 2, 3        # тоже кортеж — скобки необязательны
empty = ()         # пустой
single = (5,)      # кортеж из одного элемента — ЗАПЯТАЯ обязательна!
not_tuple = (5)    # это просто int 5, а не кортеж!
print(type((5)))   # <class 'int'>
print(type((5,)))  # <class 'tuple'>
```

Упаковка и распаковка:

```python
point = 3, 4          # упаковка
x, y = point          # распаковка
a, b = b, a           # обмен через кортеж
first, *rest = 1, 2, 3, 4   # first=1, rest=[2,3,4]
```

## «Кортежные включения» → дают генератор!

(!) Синтаксис `(expr for x in it)` создаёт **не кортеж**, а **генератор** (билет 62)! Чтобы получить кортеж из включения — оберните в `tuple()`:

```python
gen = (x**2 for x in range(3))     # это generator, НЕ кортеж!
print(type(gen))                    # <class 'generator'>

t = tuple(x**2 for x in range(3))   # (0, 1, 4) — вот кортеж
```

## Встроенный объект tuple

```python
tuple()          # ()
tuple([1, 2])    # (1, 2)
tuple("ab")      # ('a', 'b')
tuple(range(3))  # (0, 1, 2)
```

## Индексация и срезы

Как у списков, но результат среза — тоже кортеж:

```python
t = (0, 1, 2, 3, 4)
t[0]       # 0
t[-1]      # 4
t[1:3]     # (1, 2)
t[::-1]    # (4, 3, 2, 1, 0)
```

## Зачем кортежи, если есть списки

- **неизменяемость** → защита от случайных изменений, можно использовать как **ключ словаря** и элемент множества (если содержимое хешируемо);
- семантика «фиксированная запись» (например, координата `(x, y)`);
- чуть быстрее и компактнее списка.

```python
d = {(0, 0): "origin"}     # кортеж — валидный ключ
# d = {[0, 0]: "x"}        # ❌ список — нехешируем
```

## Итоги

- `tuple` — **неизменяемый** список; задаётся **запятой**;
- `(5,)` — кортеж, `(5)` — число; `()` — пустой;
- `(expr for ...)` — **генератор**, не кортеж; нужен `tuple(...)`;
- кортеж может быть ключом словаря/элементом множества (если хешируем);
- распаковка: `a, b = ...`, `first, *rest = ...`.

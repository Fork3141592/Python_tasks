# Билет 20: sort(), sorted(), аргументы reverse и key

## Две формы сортировки

| | `list.sort()` | `sorted(iterable)` |
|---|---|---|
| Кого сортирует | только список | любой итерируемый |
| Где результат | **на месте** | **новый список** |
| Возвращает | `None` | отсортированный список |

```python
s = [3, 1, 2]
s.sort()              # меняет s → [1, 2, 3], возвращает None
print(s)

t = (3, 1, 2)
result = sorted(t)    # [1, 2, 3] — новый список (исходник не тронут)
print(t)              # (3, 1, 2)
```

`s = s.sort()` обнулит `s` (вернётся `None`).

## Аргумент reverse

```python
sorted([3, 1, 2], reverse=True)   # [3, 2, 1] — по убыванию
[5, 2, 8].sort(reverse=True)       # [8, 5, 2]
```

## Аргумент key

`key` — функция, применяемая к каждому элементу; сортировка идёт по **результатам** этой функции (сами элементы не меняются):

```python
words = ["banana", "fig", "cherry"]
sorted(words, key=len)            # ['fig', 'banana', 'cherry'] — по длине

# По модулю
sorted([-3, 1, -2], key=abs)      # [1, -2, -3]

# Без учёта регистра
sorted(["B", "a", "C"], key=str.lower)   # ['a', 'B', 'C']

# Сортировка кортежей/словарей по нужному полю
people = [("Аня", 30), ("Боб", 25)]
sorted(people, key=lambda p: p[1])       # по возрасту → [('Боб',25),('Аня',30)]
```

### Сортировка по нескольким ключам

`key` возвращает кортеж — сравнение лексикографическое:

```python
data = [("a", 2), ("b", 1), ("a", 1)]
sorted(data, key=lambda x: (x[0], x[1]))
# [('a', 1), ('a', 2), ('b', 1)]

# Разные направления: число по убыванию, имя по возрастанию
sorted(data, key=lambda x: (-x[1], x[0]))
```

## Устойчивость (stable)

Сортировка в Python **устойчива**: элементы с равными ключами сохраняют исходный порядок. Это позволяет сортировать «в несколько проходов».

```python
data = [("a", 2), ("b", 2), ("c", 1)]
# сначала по имени, затем по числу — порядок имён при равном числе сохранится
res = sorted(data, key=lambda x: x[0])
res = sorted(res, key=lambda x: x[1])
```

## Итоги

- `list.sort()` — на месте, возвращает `None`; `sorted()` — новый список из любого итерируемого;
- `reverse=True` — убывание;
- `key=func` — сортировать по `func(элемент)`; кортеж в key → многоключевая сортировка;
- сортировка **устойчива** (stable).

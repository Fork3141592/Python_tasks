# Урок 5. Сортировка, копирование, кортежи

**Билеты:** 20–23

## Теория

Разбираемые билеты лежат в папке `теория/`:

- Билет 20. sort(), sorted(), аргументы reverse и key
- Билет 21. Поверхностная и глубокая копия
- Билет 22. Кортежи (tuple)
- Билет 23. Кортежи как последовательности

## Задачи

### Задача 5.1. Сортировка по длине, потом по алфавиту

`sort_words(words)` — `words` список слов. Вернуть новый список слов, отсортированный сначала по
длине, при равной длине — по алфавиту.

### Задача 5.2. Таблица лидеров

`leaderboard(rows)` — `rows` список пар `(имя, очки)`. Отсортируйте так: очки по
убыванию, а при равных очках — имена по алфавиту.

### Задача 5.3. Настоящая независимая копия

`independent_copy(matrix)` — вернуть копию списка списков, у которой **внутренние**
списки тоже независимы: изменение ячейки копии не задевает оригинал.

### Задача 5.4. Неизменяемый кортеж с изменяемым нутром

Кортеж неизменяем — нельзя переприсвоить его элемент. Но если элемент сам
изменяемый (список), его можно менять «на месте». Реализуйте
`append_to_tuple_list(t, x)`: `t` — кортеж, первый элемент которого список;
добавьте `x` в этот список и верните **тот же** кортеж.
